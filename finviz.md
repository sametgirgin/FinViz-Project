# FinVizFinance Package Overview

`finvizfinance` is a Python package designed to collect comprehensive financial information from the [FinViz](https://finviz.com) website.

## Key Features
The package provides access to several categories of financial data:

* **Stock Data:** Includes stock charts, fundamental analysis metrics, and technical information.
* **Insider Information:** Tracks insider trading activities and filings.
* **Stock News:** Fetches the latest market-related news headlines.
* **Forex:** Provides access to foreign exchange (Forex) charts.
* **Crypto:** Provides access to cryptocurrency market charts.
